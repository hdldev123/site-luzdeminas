# Pasta `public/`

Coloque aqui os assets estáticos. Substitua os placeholders por arquivos reais:

- `og-image.jpg` — imagem de compartilhamento (Open Graph) **1200×630px**.
- `apple-icon.png` — ícone para iOS **180×180px** (opcional).
- Fotos das cidades/telas referenciadas como `<<foto-...>>` / `<<screenshot-...>>`
  nos componentes — depois troque o componente `Placeholder` por `next/image`.

O favicon é gerado a partir de `src/app/icon.svg` (não precisa de arquivo aqui).
